如何制作CocoaPods的工爸本地镜像

初始化

git clone --mirror https://github.com/CocoaPods/Specs.git

首先先制作一个CocoaPods的本地镜像。大小大概有将近900M，80000多个文件，等待时间略长。

编辑config文件

之后在config文件里面配置一下我们本地镜像项目的URL。我事先在gitlab上建了一个CocoaPods-Mirror项目，将对应的地址复制过来即可。注意这里我用的是ssh地址，因为我创建的是internal的项目，访问时是需要验证的。使用ssh地址，只要将ssh key配置好，之后自动同步脚本就不用输用户名和密码了。
