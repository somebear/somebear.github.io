OS X连不上wifi

Posted on 2014-03-10   |  

今天Macbook连不上wifi了，一开始是看不到自己的路由器只能看到其他的人网络，后来能看到了但是连不上去。

google了一下找到的解决方案：

重启并在启动时同时按下 command + option + p + r ，启动音响(”dang~”)三次之后即可。

这个组合键的作用是用来重置NVRAM和PRAM的。

关于NVRAM和PRAM的内容摘自Apple Support:

NVRAM/PRAM: Mac 会将某些设置存储在特定的存储区中，即使关机这些设置也不会丢失。在基于 Intel 的 Mac 上，存储位置是称为 NVRAM 的内存；而在基于 PowerPC 的 Mac 上，存储位置则是称为 PRAM 的内存。
存储在 NVRAM/PRAM 中的信息包括：

  • 扬声器音量

  • 屏幕分辨率

  • 启动磁盘选择

  • 最近的内核崩溃信息（如果有的话）

如果遇到的问题与上述功能相关，您可能需要重置 NVRAM 或 PRAM。例如，如果 Mac 并非从“启动磁盘”偏好设置中指定的启动磁盘启动，或者启动 Mac 时暂时出现了“问号”图标，重置 NVRAM 或 PRAM 可能会有帮助。

注：OS X 不会将网络设置存储在 NVRAM/PRAM 中。如果您正在故障诊断网络问题，重置不会有任何帮助。

虽然官方文档说对网络故障没有任何帮助，但是却确实解决了我的问题。。。
