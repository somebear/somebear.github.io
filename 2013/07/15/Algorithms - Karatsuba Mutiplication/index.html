Algorithms - Karatsuba Mutiplication

發表於 2013-07-15   |  

算法简介

Karatsuba Mutiplication算法可以快速的将两个n位的整数的乘积算出。对于大整数来说计算速度优于学校教授的传统计算方法。

算法步骤

预备

将n位的整数平均分为前后两个部分，假设第一个整数前半部分为a，后半部分为b，第二个整数分为c和d。

步骤

  1. 计算a*c = r1
  2. 计算b*d = r2
  3. 计算(a+b) * (c+d) = r3
  4. 计算r3 - r2 - r1 = r4
  5. 计算r1*10^n + r2 + r4*10^(n/2)，即得出结果。

例子

x = 5678
y = 1234
计算x*y (结果:7006652)

将x均分成a和b: a= 56, b= 78
将y均分成c和d: c= 12, d= 34

r1 = a*c = 56*12 = 672
r2 = b*d = 78*34 = 2652
r3 = (a+b)*(c+d) = (56+78)*(12+34) = 134*46 = 6164
r4 = r3 - r2 - r1 = 6164 - 2652 - 672 = 2840
计算结果为 672*10^4 + 2652 + 2840*10^2 = 6720000 + 2652 + 284000 = 7006652
