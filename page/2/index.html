Algorithms - Karatsuba Mutiplication

发表于 2013-07-15

算法简介

Karatsuba Mutiplication算法可以快速的将两个n位的整数的乘积算出。对于大整数来说计算速度优于学校教授的传统计算方法。

算法步骤

预备

将n位的整数平均分为前后两个部分，假设第一个整数前半部分为a，后半部分为b，第二个整数分为c和d。

步骤

  1. 计算a*c = r1
  2. 计算b*d = r2
  3. 计算(a+b) * (c+d) = r3
  4. 计算r3 - r2 - r1 = r4
  5. 计算r1*10^n + r2 + r4*10^(n/2)，即得出结果。

例子

x = 5678
y = 1234
计算x*y (结果:7006652)

将x均分成a和b: a= 56, b= 78
将y均分成c和d: c= 12, d= 34

r1 = a*c = 56*12 = 672
r2 = b*d = 78*34 = 2652
r3 = (a+b)*(c+d) = (56+78)*(12+34) = 134*46 = 6164
r4 = r3 - r2 - r1 = 6164 - 2652 - 672 = 2840
计算结果为 672*10^4 + 2652 + 2840*10^2 = 6720000 + 2652 + 284000 = 7006652

最近看Python Cookbook感觉进度有些慢，就打算开始看《Python参考手册》。由于Cookbook是一些经典的例子和代码就留着以后慢慢看啦。
下面是看参考手册时遇到的值得积累的知识点:

第一章

  • 用交互方式使用Python时，特殊变量_用于保存最后一次运算的结果。
  • 对于Python文件来说，解释器会不断运行直到到达输入文件的结尾。如果是以交互方式运行，有两种方法可以退出解释器，一种是输入EOF(End of File，文件结束)，另一种是用Python IDE的下拉菜单中选择exit。
  • format()函数使用的格式说明符类似与传统字符串格式化运算符(%)使用的格式说明符，比如:
    print “{0:3d} {1:0.2f}”.format(year,principal)
    其中，”0:3d”和”1:0.2f”中冒号前的数字代表传给format方法的相关参数，而冒号后面的部分则是格式说明符。
  • print后面跟上’,’将忽略换行符。
