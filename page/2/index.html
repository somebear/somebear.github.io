Algorithms - Karatsuba Mutiplication

發表於 2013-07-15   |  

算法简介

Karatsuba Mutiplication算法可以快速的将两个n位的整数的乘积算出。对于大整数来说计算速度优于学校教授的传统计算方法。

算法步骤

预备

将n位的整数平均分为前后两个部分，假设第一个整数前半部分为a，后半部分为b，第二个整数分为c和d。

步骤

  1. 计算a*c = r1
  2. 计算b*d = r2
  3. 计算(a+b) * (c+d) = r3
  4. 计算r3 - r2 - r1 = r4
  5. 计算r1*10^n + r2 + r4*10^(n/2)，即得出结果。

例子

x = 5678
y = 1234
计算x*y (结果:7006652)

将x均分成a和b: a= 56, b= 78
将y均分成c和d: c= 12, d= 34

r1 = a*c = 56*12 = 672
r2 = b*d = 78*34 = 2652
r3 = (a+b)*(c+d) = (56+78)*(12+34) = 134*46 = 6164
r4 = r3 - r2 - r1 = 6164 - 2652 - 672 = 2840
计算结果为 672*10^4 + 2652 + 2840*10^2 = 6720000 + 2652 + 284000 = 7006652

到此我们需要计算4个乘法即ac, ad, bc, bd。这时如果发现a*c仍然是大整数相乘，还可以将a*c用Karatsuba算法计算，同理如果a*c的乘法子项依然过大我们还可以继续使用该算法计算，因此在程序实现阶段我们是用的是递归。由于递归的函数的主体是乘法部分所以我们应该尽量减少乘法的次数。于是有了优化的版本，因为我们已经有了ac和bd的值，ad+bc就可以用(a+b)*(c+d)-ac-bd来算出，即步骤3和步骤4做的事情:
